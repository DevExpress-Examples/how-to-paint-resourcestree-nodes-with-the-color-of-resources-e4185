Developer Express Example
How to paint ResourcesTree nodes with the color of resources

This example illustrates how to customize ResourcesTree (http://documentation.devexpress.com/#WindowsForms/clsDevExpressXtraSchedulerUIResourcesTreetopic) nodes painting so that its nodes have the same color as corresponding resources. First of all, it should be especially noted that there is no appropriate method to achieve this result when default resource color schemas are used (see SchedulerControl.ResourceColorSchemas Property (http://documentation.devexpress.com/#WindowsForms/DevExpressXtraSchedulerSchedulerControl_ResourceColorSchemastopic)) because their indexes might not correspond resource indexes (especially if a complex master-detail hierarchy of resource is used). So, you need to specify resource colors explicitely. In this example, we have mapped the Resource.Color Property (http://documentation.devexpress.com/#CoreLibraries/DevExpressXtraSchedulerResource_Colortopic) to the "Color" column in "Resources" table. Then, we retrieve the current resource color in the TreeList.CustomDrawNodeCell Event (http://documentation.devexpress.com/#WindowsForms/DevExpressXtraTreeListTreeList_CustomDrawNodeCelltopic) and assign it to the e.Appearance.BackColor parameter of this event.
